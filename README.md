# CE-OptimizedLoss

[![License](https://img.shields.io/badge/License-Apache%202.0-brightgreen.svg)](https://opensource.org/licenses/Apache-2.0)

Computes the MWER (minimum WER) Loss with beam search and negative sampling strategy.
